# Airy-Quality-Sensor

## Overview

This project is an air quality sensor using Raspberry Pi.

### Goals and Aims
- Detect air pollutants and air freshener chemicals
- Apply machine learning and data techniques for classification
- Securely transmit and manage sensor information

| Role | Responsibilty |
|------|---------------|
| Computer Science | backend and Raspberry Pi programming |
| Data Science | Data processing |
| Cyber Security | secure communication |
